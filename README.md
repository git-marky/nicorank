# nicorank

ニコニコランキングメーカーは「ランキング動画」を作成するためのソフトです。

# Features

ニコニコ動画へ投稿するための「ランキング動画」作成に特化しています。
ニコニコ動画の再生数やマイリスト数などのデータの集計から、動画ダウンロード、変換、出力までを支援します。
